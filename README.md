# Flashcard App

This project is a small flashcard application built with React and Vite.

## Setup

1. Copy `.env.example` to `.env` and provide your Firebase credentials.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Build

To create a production build, run:
```bash
npm run build
```

Fonts are loaded via `@fontsource/roboto`. Unused TTF files in `public/fonts` have been removed to reduce bundle size.
